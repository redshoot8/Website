Проекты

Telegram PhotoBot

PhotoBot - это мой Бот для Telegram, который угадывает объекты на изображении. В него интегрирована предварительно обученная модель для классификации изображений.

Репозиторий на GitHub: PhotoBot

PhotoEditor

PhotoEditor - это моя программа, которая позволяет производить базовые процедуры обработки изображения. С ее помощью можно применить к изображению фильтры, вклеить одно изображение на второе, нарисовать что-то на изображении и т.д.

Репозиторий на GitHub: PhotoEditorProject

Random Number Image Generator Website

Random Number Image Generator Website - это мой вебсайт, который генерирует изображения со случайными числами заданной длины. В него интегрирована моя предварительно обученная модель по генерации изображений рукописных цифр.

Репозиторий на GitHub: RandNumImgGeneratorWebsite
